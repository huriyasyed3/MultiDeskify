# 🖥️ MultiDeskify

Tauri-based desktop app for running multiple apps in native webviews.

---

## 🚀 Run
npm install
npm run tauri dev

---

## 🏗️ Build
npm run tauri build

Output:
src-tauri/target/release/bundle

---

## 🧪 Test
- Switch apps → webview open
- Multiple apps → independent run
- Smooth switching

---

## 🛠️ Tech
React • Tauri • Rust • Node.js
